# Calculadora Flex

O Projeto da calculadora flex foi propsta do Microfundamento de Desenvolvimento de aplicações mobile do curso de Análise e Desenvolvimento de Sistemas da Puc Minas Virtual, ministrado pelo professor Kleber Souza.

## Aula 01
Definição das páginas, importação do bottom navigation do react native paper. Criação das páginas e implementação da navegação entre a calculadora e a lista de gastos.

## Aula 02
Implementação da visualização em lista dos dados de abastecimento e do botão, ainda sem funcionalidade, para cadastro de novo registro.

## Aula 03
Acréscimo do NavigationContainer para criar uma pilha de telas, melhorando as opções de navegação. Foi implementado o acesso à página "Abastecimento" através de um clique no FAB da tela de gastos.

## Aula 04
Implementação da função goBack para possibilitar desempilhar telas, voltando de Abastecimento para a anterior (Gastos).

## Aula 05
Desenvolvimento do formulário para cadastro de novo abastecimento. Implementação dos botões tipo radio para a definição do tipo de combustível.

## Aula 06
Acréscimo dos inputs de dados e dos botões de excluir e salvar da página abastecimento.

## Aula 07
Importando a biblioteca DateTimePicker. Acréscimo do TouchableOpacity para permitir abrir o calendário ao clicar no campo de input da data. Importação da biblioteca Moment para a formatação da data.

## Aula 08
Modificando os componentes para que as informações de um abastecimento sejam carregadas na tela 

## Aula 09
Uso do hook useEffect para a exibição das informações de um abastecimento na tela carregada, viabilizando edição.
