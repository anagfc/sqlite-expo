# Calculadora Flex

O Projeto da calculadora flex foi propsta do Microfundamento de Desenvolvimento de aplicações mobile do curso de Análise e Desenvolvimento de Sistemas da Puc Minas Virtual, ministrado pelo professor Kleber Souza.

## Aula 01
Foi utilizado o React Native Paper para a criação do header da aplicação.

## Aula 02
Acréscimo do TextInput para a obtenção de dados e do Button para indicar a ação de cálculo que será implementada posteriormente.

## Aula 03
Ajustes visuais do layout e inserção do campo texto que receberá a resposta.

## Aula 04
Início da refatoração: eliminação dos códigos repetidos. Com a criação da pasta 'components', tudo o que será reaproveitado no código ganha espaço próprio e que viabiliza a reutilização. Criação dos componentes Container, Header e Body.

## Aula 05
Criação da lógica de negócio e vinculação das respostas.
